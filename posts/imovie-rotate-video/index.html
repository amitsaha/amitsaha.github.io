Tip Rotating video frames using iMovie

January 1, 2018

I had to rotate a few video frames using iMovie. The instructions for rotating the frames were working, but the rotation was being applied to “neighboring” frames as well. It turns out the trick is to use “split clip” to split a frame so that you can isolate the exact frame to be rotated. And that worked!

Thanks to this forum reply.
