I want the equivalent of ternary operator ? (in C or javascript) in Python. I have forgotten exactly how, but i know that i can use if..else in list comprehension, so perhaps it works outside it too?

Let’s try:

>>> import random
>>> value = 1 if random.random() > 0.5 else 0
>>> value
0
>>> value = 1 if random.random() > 0.5 else 0
>>> value
0
>>> value = 1 if random.random() > 0.5 else 0
>>> value
1

yeah, it does.

of course, random.random() > 0.5 is an example of a conditional evaluation.
