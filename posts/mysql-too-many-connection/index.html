Tip MySQL - Too many connections

2018-01-04 (Updated: 2019-06-17)
mysql

If you are getting the “Too many connections” error, couple of things worth checking on the MySQL server:

mysql> show processlist;
..

The above will show the currently open connections. The second is:

mysql> show variables like "max_connections";
..

The above will show the configured max_connections allowed. When you get this error, this value is less than or equal to the number of rows returned by show processlist.
